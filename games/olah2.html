Olah 2 is the successor to Olah, the first real game I ever created. The sequel improved on every aspect of the original, and added features that we simply couldn't have implemented while writing the original. It added seamless keyboard support that allowed for simulatious keyboard and controller players! Somehow, we were able to truly surpass expectations for a sequel to a game that had already surpassed expectations.

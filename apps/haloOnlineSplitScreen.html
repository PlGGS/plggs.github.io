Halo Online Split Screen, otherwise known as HOSS, is an application I wrote in order to allow players of ElDewrito to play with up to three other people on one PC. This was the first application I wrote to solve a real problem in C#. After it's release, the developers of the the actual game added features that will be present in the next version that will allow Halo Online Split Screen to take up much less storage space that is does right now.

        /// </summary>

        /// <param name="shortcutIconLocation"></param>
        /// <param name="shortcutDescription"></param>
        public static void CreateShortcut(string shortcutName, string shortcutPath, string shortcutTargetFileLocation, string shortcutArguments, string shortcutStartInDirectory, string shortcutIconLocation, string shortcutDescription)
        {
            
            string shortcutLocation = System.IO.Path.Combine(shortcutPath, shortcutName + ".lnk");
            WshShell shell = new WshShell();
            IWshShortcut shortcut = (IWshShortcut)shell.CreateShortcut(shortcutLocation);

            shortcut.Description = shortcutDescription;
            shortcut.IconLocation = shortcutIconLocation;
            shortcut.TargetPath = shortcutTargetFileLocation;
            shortcut.Arguments = shortcutArguments;
            shortcut.WorkingDirectory = shortcutStartInDirectory;
            shortcut.Save();

        }

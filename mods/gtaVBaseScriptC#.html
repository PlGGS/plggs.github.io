This base script is just that. It's a file that people can use to create GTA V mods in C#! It's what I've based a couple mods off of and a great resource for those looking into GTA V modding. I've included a link to ScriptHookV's GitHub page to make sure people aren't downloading sketchy ScriptHookV DLLs!
